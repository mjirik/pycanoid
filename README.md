========


    sudo apt-get install python-pygame python opencv

Run
===

   python pycanoidMain.py

Calibration
===========

Posuňte obličej na požadovanou hranici a stiskněte:
4 - Levá hranice
6 - Pravá hranice
8 - Horní hranice
2 - Spodní hranice

Pro ukončení stiskněte ESC.
